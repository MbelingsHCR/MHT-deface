~01.Tebas Index
~02.Web Scan
~03.Auto Deface
~04.Tools Installer
~05.Bikin SC Deface

Fungsi Tools Defacer:
1). Tebas Index (Berfungsi untuk Deface halaman depan website secara langsung)

2). Web Scan (Berfungsi untuk mengetahui informasi website seperti alamat IP,subdomain,vuln sqli,Dll)

3). Auto Deface (Berfungsi untuk men Deface website secara otomatis sebanyak 10 website lebih)

4). Tools Installer (Alat penyedia install secara otomatis beberapa alat hacking dengan cara yg instan)

5). Bikin Script Deface (Berfungsi untuk membuat script Deface dengan mudah dan simple di Termux)

6). Shell Finder (Berfungsi untuk mencari sebuah akses shell backdor orang lain dengan menggunakan metode Brute Force)
